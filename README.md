# 3D_Game-6  
## 智能巡逻兵  
### 提交要求：  
- 游戏设计要求：  
  - 创建一个地图和若干巡逻兵(使用动画)；  
  - 每个巡逻兵走一个3~5个边的凸多边型，位置数据是相对地址。即每次确定下一个目标位置，用自己当前位置为原点计算；  
  - 巡逻兵碰撞到障碍物，则会自动选下一个点为目标；  
  - 巡逻兵在设定范围内感知到玩家，会自动追击玩家；  
  - 失去玩家目标后，继续巡逻；  
  - 计分：玩家每次甩掉一个巡逻兵计一分，与巡逻兵碰撞游戏结束  
- 游戏规则：  
  - 每一次躲开巡逻兵进入下一个巡逻兵的范围，会加分  
  - 初始情况只有四个巡逻兵，每
